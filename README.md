# Rocky Linux 9 - Security Labs

Projet pratique de cybersécurité Linux basé sur Rocky Linux 9.

## Objectifs

- Durcissement système (CIS Level 1)
- Sécurité réseau
- SELinux enforcing
- Audit et conformité
- Automatisation via Ansible

## Environnement

- Rocky Linux 9 minimal
- Accès root
- Usage pédagogique et professionnel

## Contenu

- Ansible : automatisation sécurité
- Scripts Bash : hardening initial
- Documentation technique
- Rapport OpenSCAP CIS

## Niveau

Linux Security Engineer / DevSecOps
